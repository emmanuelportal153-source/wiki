¿Qué es una wiki colaborativa?

Una wiki es un sitio web donde varias personas pueden escribir, editar y actualizar información de forma sencilla. La idea es que el contenido se construye entre todos: alguien crea una página, otros la corrigen, agregan ejemplos o actualizan los datos.

Las wikis se usan mucho para compartir conocimientos, hacer resúmenes, explicar temas de clase o documentar información importante dentro de un grupo o una comunidad.

¿Para qué sirve una wiki?

Una wiki sirve para organizar información de manera clara y que cualquier persona del equipo pueda participar. Se pueden crear páginas con conceptos, ejemplos, pasos de un proceso o materiales de apoyo. Cada página se puede editar desde el navegador sin tener que saber programación.

Otra ventaja es que normalmente se guarda un historial de cambios. Esto permite ver quién modificó qué, corregir errores y recuperar versiones anteriores en caso de que algo salga mal.

¿Qué debería incluir tu propia página de wiki?

Si quieres crear una página similar para otro tema, es recomendable que siempre incluyas:

  • Un título claro del tema.
  • Una explicación breve y directa.
  • Al menos una imagen que represente el contenido.
  • Algún video o recurso extra que complemente la información.
  • Tu nombre o el de tu equipo como creadores.
  • Referencias básicas de dónde obtuviste la información.
